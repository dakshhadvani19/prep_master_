# **This is an AI powered University exam preparation tool to make students score more in their exams.**

## **For what it usefull for** 🤔

🚀 It gives ability to predict and practise more accurate questions for university exams.

🎯 Made for university students .

Key Features : Robust accuracy , Syallbus relative , Review your tests , Easy to Use ✅

Live Here 👇🏼
🌐 https://prepmaster-tau.vercel.app/

## **Quick Start Guide** ⚡

• Create an account .

• Direct to Home / Landing page .

• Select your domain , course .

• Look for you semester , subject and then you can start your test .

• Upload Your supported files to get questions only from that Topic .

##**Why this is different than AI's like Notebook lm , ChatGPT and etc?**💡

• Accurate exam practise 

• Use univerisity syallabus files to generate questions 

## **Tech Stack**

• FrontEnd -> ReactJS , Vanila CSS ( for more better animation )

• BackEnd -> Node.Js , Next.Js ( Somewhere )

• Database -> Firebase ( currently )
